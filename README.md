# Shadowfox_Advanced_level_task
# Advanced Cricket Fielding Analytics using Python

## Project Overview

This project develops an advanced cricket fielding analytics system to evaluate player performance using event-based fielding data. It was completed as part of the **Advanced Level Data Science Internship** at **ShadowFox**.

The project applies data processing, performance scoring, ranking, and Excel report generation using Python.

---

## Objectives

- Analyze player fielding performance
- Calculate performance scores using weighted metrics
- Rank players based on fielding efficiency
- Generate qualitative player insights
- Export results into a professionally formatted Excel workbook

---

## Technologies Used

- Python
- Pandas
- OpenPyXL
- Logging

---

## Features

- Ball-by-ball fielding event analysis
- Performance score calculation using weighted metrics
- Player ranking system
- Qualitative performance insights
- Automated Excel workbook generation
- Professional formatting with multiple worksheets
- Dynamic column sizing and styling

---

## Performance Metrics

The analytics engine evaluates players using:

- Clean Picks (CP)
- Good Throws (GT)
- Catches (C)
- Dropped Catches (DC)
- Stumpings (ST)
- Run Outs (RO)
- Missed Run Outs (MRO)
- Direct Hits (DH)
- Runs Saved (RS)
- Overall Performance Score (PS)

---

## Project Workflow

1. Load Ball-by-Ball Fielding Data
2. Process Player Statistics
3. Calculate Performance Scores
4. Generate Player Rankings
5. Create Qualitative Insights
6. Export Professional Excel Report
